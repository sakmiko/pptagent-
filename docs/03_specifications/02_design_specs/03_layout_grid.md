# 4. 布局网格规范

布局网格是确保页面元素对齐、创造视觉秩序和提升专业感的无形框架。本规范为 AutoPPT Agent 的所有页面定义了统一的画布、安全边距和栅格系统。

## 4.1 画布属性

- **宽高比 (Aspect Ratio)**: 所有页面的画布宽高比严格固定为 **16:9**。这是现代演示文稿和宽屏显示器的标准比例，能提供最佳的视觉体验。
- **尺寸**: 在 PowerPoint 中，这通常对应于 `10英寸 x 5.625英寸` 或 `33.867厘米 x 19.05厘米` 的尺寸。

## 4.2 安全边距 (Safe Margins)

为了防止文本或重要视觉元素过于贴近屏幕边缘而被裁剪或显得拥挤，所有内容都必须被放置在安全边距之内。

- **左右边距**: **0.35 英寸**
- **上下边距**: **0.25 英寸**

渲染引擎在放置任何文本框或图片占位符时，都必须计算并遵守这些边距。

## 4.3 基础栅格系统

虽然在 PPT 环境中实现复杂的栅格系统有一定限制，但我们可以遵循其核心原则来指导布局。我们采用一个简化的、基于列的栅格概念。

- **主内容区**: 除去安全边距后，页面中央的可操作区域。
- **列（Columns）**: 我们可以将主内容区虚拟地划分为 **12** 列。这为布局提供了极大的灵活性。
  - **双栏布局 (`section_two_col`)**: 每栏可以占据 6 列，中间留有适当的间距（Gutter）。
  - **图文布局 (`text_image_split`)**: 文本和图片可以各占 6 列，或者采用 5-7 或 7-5 的不对称布局以创造视觉动感。
  - **四宫格布局 (`section_four_grid`)**: 可以看作是两行两列的布局，每格占据 6 列的宽度。

## 4.4 垂直节奏 (Vertical Rhythm)

垂直方向上的对齐和间距同样重要。我们定义一个基础的垂直间距单位，以确保元素在垂直方向上保持一致的节奏感。

- **基础单位**: `0.18 英寸`
- **应用**: 
  - **标题与正文间距**: 建议为 1 个基础单位 (`0.18英寸`)。
  - **段落间距**: 建议为 0.5 个基础单位 (`0.09英寸`)。
  - **列表项间距**: 建议为 0.25 个基础单位 (`0.045英寸`)。

## 4.5 应用示例

以 `section_two_col`（双栏文本页）为例，其布局应遵循以下规则：

1.  **主标题 (`title`)**: 顶部对齐安全边距上边缘，左右居中。
2.  **左栏容器 (`text_l`)**: 左侧对齐左安全边距，顶部与主标题保持 `0.18英寸` 的间距。其宽度约等于 `(主内容区宽度 - 间距) / 2`。
3.  **右栏容器 (`text_r`)**: 右侧对齐右安全边距，其顶部与左栏容器的顶部对齐。其宽度与左栏相同。
4.  **栏间距 (Gutter)**: 左右两栏之间应留有足够的空白，建议宽度为 `0.3 - 0.5英寸`。

通过严格遵循这些布局和网格规范，AutoPPT Agent 能够生成具有专业设计感的、视觉上和谐统一的演示文稿。
